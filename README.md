# 블로그치트키 (BlogCheatKey)

**AI 기반 SEO 최적화 블로그 콘텐츠 자동 생성 플랫폼**

> 키워드 하나만 입력하면, AI가 SEO 조건을 완벽히 충족하는 고품질 블로그 글을 자동으로 작성합니다.

---

## 🎯 핵심 가치

| 문제 | 해결책 |
|------|--------|
| SEO 최적화된 블로그 작성에 **전문 지식 필요** | AI가 키워드 분석부터 최적화까지 **자동 처리** |
| 양질의 콘텐츠 제작에 **3-4시간 소요** | **3분 만에** 전문가 수준의 글 완성 |
| 형태소 빈도, 글자수 등 **SEO 조건 맞추기 어려움** | **15-17회 형태소 출현**, **1,700-2,000자** 자동 조절 |

---

## ✨ 주요 기능

### 🔍 AI 키워드 분석
- 검색 의도 자동 파악
- SEO 최적화 소제목 4개 자동 제안
- 사용자 고민 포인트 분석

### 📝 SEO 최적화 콘텐츠 생성
- **키워드 형태소 분석**: 각 형태소 15-17회 정확한 출현 빈도 관리
- **글자수 최적화**: 1,700-2,000자 (공백 제외)
- **구조화된 콘텐츠**: 서론(35%) → 본론(4개 섹션) → 결론

### 📊 실시간 연구 데이터 수집
- 신뢰할 수 있는 정보원에서 최신 데이터 수집
- 연구 자료 기반 설득력 있는 콘텐츠
- 출처 자동 인용

### 💬 AI 챗봇 편집 (프리미엄)
- 자연어로 콘텐츠 수정 요청
- **SSR 평가 기반 Top 5 제목 추천** (25가지 스타일 → AI 평가 → 상위 5개)
- 톤앤매너, 설득력, 가독성 조정
- 실시간 SEO 조건 재검증

### 📱 모바일 최적화 포맷팅
- 25-30자 단위 자동 줄바꿈 (규칙 기반)
- AI 스마트 포맷팅 (문맥 이해 기반)
- 원클릭 복사 기능

### 💾 세션 관리
- 작업 내용 자동/수동 저장
- 이전 세션 불러오기
- 프로젝트 히스토리

### 🖼️ 이미지 생성 안내
- 이미지 요청 시 **외부 도구 안내**
- 추천: [Google Whisk](https://labs.google/fx/tools/whisk), [Napkin AI](https://www.napkin.ai/)

## 🔄 사용 워크플로우

```
1. 키워드 입력 (예: "벤츠엔진경고등")
       ↓
2. AI 키워드 분석 → 검색 의도 + 소제목 4개 제안
       ↓
3. 실시간 연구 데이터 수집 (Perplexity)
       ↓
4. 비즈니스 정보 입력 (업종, 전문성, 차별화 요소)
       ↓
5. SEO 최적화 콘텐츠 자동 생성 (Claude)
       ↓
6. AI 챗봇으로 편집/제목 추천 (프리미엄)
       ↓
7. 모바일 최적화 포맷 → 복사 → 블로그 발행
```

## 📊 SEO 최적화 기준

| 항목 | 기준 |
|------|------|
| **완전 키워드 출현** | 5-7회 |
| **개별 형태소 출현** | 각 15-17회 (자동 조절) |
| **글자수** | 1,700-2,000자 (공백 제외) |
| **구조** | 서론 35% + 본론 4섹션 + 결론 |
| **과다 사용 방지** | 18회 초과 시 동의어 자동 치환 |

### 🇰🇷 한국어 형태소 분석 엔진
- `hangul-js`, `korean-js` 라이브러리 활용
- 조사/어미 정확한 분리 (예: "벤츠엔진경고등에서는" → "벤츠엔진경고등")
- 30개 이상의 한국어 조사/어미 패턴 지원

## 🛠 기술 스택

### Frontend
- **React 18** + TypeScript
- **Vite** - 빌드 도구
- **TailwindCSS** + **Radix UI** - UI 컴포넌트
- **TanStack Query** - 서버 상태 관리
- **Wouter** - 라우팅
- **Framer Motion** - 애니메이션

### Backend
- **Node.js** + **Express** - REST API
- **TypeScript** - 타입 안전성
- **PostgreSQL** + **Drizzle ORM** - 데이터베이스
- **Session 관리** - PostgreSQL 세션 스토어

### 인증 시스템
- **소셜 로그인**: Google, Kakao, Naver OAuth
- **이메일/비밀번호** 로그인
- **관리자 대시보드** (권한 관리, 사용자 관리)

### 결제 연동
- **포트원(PortOne)** - 카드/계좌이체 결제
- **무통장 입금** - 카카오뱅크 계좌이체 후 수동 권한 부여
